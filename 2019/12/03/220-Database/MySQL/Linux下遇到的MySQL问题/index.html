220-Database/MySQL/Linux下遇到的MySQL问题

2019-12-03

The Server quit without updating PID file

在启动 mysql 服务时，提示下面错误

[root@Centos6 etc]# service mysql start
Starting MySQL.The server quit without updating PID file (/usr/local/mysql/data/Centos6.5.pid).

解决方法：

最最关键的是查看日志文件： /usr/local/mysql/data/xxx.err
